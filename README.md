# Thread Scanner (Read-Only Reddit Analysis Tool)

This project is a simple Python-based tool that retrieves publicly available Reddit posts using the Reddit API.

## Purpose

The tool is designed to help identify active and relevant discussions across selected subreddits for manual review. It supports more thoughtful and context-aware human participation by surfacing discussions that may benefit from meaningful engagement.

## Functionality

The tool performs read-only operations and collects basic metadata from Reddit posts, including:

- Title
- Subreddit
- Score (upvotes)
- Number of comments
- Post age
- Permalink

This information is used to organize and review discussions.

## Important Notes

This tool does NOT:

- Submit posts
- Submit comments
- Vote on content
- Send messages
- Automate any interaction with Reddit users

All participation on Reddit is performed manually by a human user.

## Technology

- Python
- PRAW (Python Reddit API Wrapper)

## Status

This project is in early-stage development and is intended for internal use.
